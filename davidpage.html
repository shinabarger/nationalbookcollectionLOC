Comparing Books in the National Collection

The Assessing the Physical Condition of the National Collection (ANC) project compares the physical, chemical and optical characteristics of 500 “identical” books from five large research libraries in distinct regions of the United States.This tool allows you to compare books used in the ANC project side-by-side.

Development Approach

SB Admin 2 makes extensive use of Bootstrap 4 utility classes in order to reduce CSS bloat and poor page performance. Custom CSS classes are used to create custom components and custom utility classes.

Before working with this theme, you should become familiar with the Bootstrap framework, especially the utility classes.
